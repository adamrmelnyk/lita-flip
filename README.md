# lita-flip

A handler for [Lita][1] that angrily flips your text.

## Installation

Add lita-flip to your Lita instance's Gemfile:

``` ruby
gem "lita-flip"
```

## Usage

Lita: flip mondays
> (╯°□°）╯︵sʎɐpuoɯ
